## Запуск приложения

Для запуска приложения выполните следующие шаги:

1. Установите зависимости:
```
npm install
```

2. Установите адрес сервера в .env (смотри .env-example)

3. Запустите сервер разработки:
```
npm run dev
```

4. Откройте браузер и перейдите по адресу:

```
http://localhost:5173/
```
